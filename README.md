# CodeStack – Backend (Node.js / Express)

CodeStack Backend provides all server-side features such as authentication, roles, courses, transactions, course progress, and dashboard analytics.

## Backend Live

https://lms-backend-s4fp.onrender.com

## Tech Stack

Node.js, Express.js, MongoDB, Mongoose, JWT, Cloudinary

## Features

- User Signup & Login
- JWT Authentication
- Role-Based Access (Student / Instructor)
- Course Management
- Course Purchase & Transactions
- Purchase History
- Lesson Progress
- Certificate Generation
- Dashboard Statistics
- Secure API Routes

## Folder Structure

server/  
 ├── controllers/  
 ├── models/  
 ├── routes/  
 ├── middleware/  
 ├── utils/  
 └── index.js

## Clone

git clone https://github.com/Abhishek-sharma-project/lms-backend
cd backend

## Install

npm install

## Environment

Create a `.env` file:  
MONGO_URI=your-mongo-uri  
JWT_SECRET=your-secret  
PAYMENT_KEY=your-payment-key

## Start

node index.js
